Пиццерия — мобильное приложение для заказа пиццы
Описание
Мобильное приложение для заказа пиццы с выбором размера, дополнительных ингредиентов и корзиной. Разработано на .NET MAUI.

Функционал
1. Просмотр меню пицц

2. Выбор размера (Маленькая / Средняя / Большая)

3. Выбор дополнительных ингредиентов

4. Динамический расчёт стоимости

5. Добавление в корзину

6. Управление корзиной (изменение количества, удаление)

7. Оформление заказа с валидацией данных

Экраны
Меню — список пицц с кнопкой выбора

Конструктор — выбор размера, ингредиентов, количества

Корзина — список товаров, итоговая сумма, оформление

Оформление — форма с данными клиента (опционально)

Технологии
.NET MAUI 9.0

MVVM

JSON (хранение меню и ингредиентов)

Shell навигация

Установка и запуск
Требования
Visual Studio 2022 с нагрузкой .NET MAUI

Windows 10/11 или Android/iOS эмулятор

Запуск
1. Клонируйте репозиторий

2. Откройте PizzeriaApp.sln

3. Выберите Windows Machine (или Android)

4. Нажмите F5

Данные
Меню и ингредиенты загружаются из Resources/Raw/menu_data.json. При отсутствии файла используются встроенные данные.
