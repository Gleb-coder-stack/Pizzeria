**Пиццерия — мобильное приложение для заказа пиццы**
Описание
Мобильное приложение для заказа пиццы с выбором размера, дополнительных ингредиентов и корзиной. Разработано на .NET MAUI.

Экраны
Меню — список пицц с кнопкой выбора
Конструктор — выбор размера, ингредиентов, количества
Корзина — список товаров, итоговая сумма, оформление
Оформление — форма с данными клиента (опционально)

Технологии
.NET MAUI 
MVVM
JSON (хранение меню и ингредиентов)
Shell навигация

Запуск
1. Клонируйте репозиторий
2. Откройте PizzeriaApp.sln
3. Выберите Windows Machine (или Android)
4. Нажмите F5

Данные
Меню и ингредиенты загружаются из Resources/Raw/menu_data.json.
